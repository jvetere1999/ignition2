# Mobile App

Access Passion OS on iPhone and iPad.

## Installation

### Add to Home Screen

1. Open Safari on iOS/iPadOS
2. Navigate to passion-os.ecent.online
3. Tap the Share button
4. Select "Add to Home Screen"
5. Name it "Passion OS"
6. Tap "Add"

The app will appear on your home screen as a native-like icon.

## Mobile Interface

The mobile version (`/m`) features:
- **Bottom Tab Navigation**
  - Home
  - Focus
  - Quests
  - Progress
  - More

### Optimizations
- Larger touch targets
- Safe area handling (notch/home indicator)
- Faster load times
- Offline support

## Features

### Available on Mobile
- Focus timer (full functionality)
- Quest viewing and progress
- Progress and XP tracking
- Sign in/out

### Desktop Only
- Full Planner calendar
- Arrange (Piano Roll/Drums)
- Shortcuts Hub
- Reference Tracks
- Admin Console

## Offline Support

The PWA works offline for:
- Viewing cached content
- Starting focus sessions
- Basic navigation

When online again:
- Data syncs automatically
- Pending actions complete

## Notifications

On iOS 16.4+, web push notifications work:
1. Add to Home Screen first
2. Enable notifications when prompted
3. Receive reminders for:
   - Focus sessions
   - Scheduled workouts
   - Quest deadlines

## Tips

1. **Add to Home Screen** - Best experience
2. **Use Safari** - Other browsers have limitations
3. **Keep updated** - Refresh occasionally for updates
4. **Desktop for production** - Use desktop for music tools
