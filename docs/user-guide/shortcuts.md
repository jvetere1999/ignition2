# Keyboard Shortcuts

Navigate Passion OS quickly with keyboard shortcuts.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Esc` | Close modal/palette |

## Command Palette

The Command Palette provides quick access to everything:

### Opening
Press `Cmd + K` (Mac) or `Ctrl + K` (Windows/Linux)

### Using
1. Type to search
2. Use arrow keys to navigate
3. Press Enter to select
4. Press Esc to close

### Available Commands
- Navigate to any page
- Start focus session
- Open settings
- Toggle theme
- Search entries

### Prefix Commands
- `>` - Run commands
- `/` - Navigate to page
- `@` - Search entries

## Focus Page

| Shortcut | Action |
|----------|--------|
| `Space` | Start/Pause timer |
| `R` | Reset timer |
| `N` | Next mode |

## Planner

| Shortcut | Action |
|----------|--------|
| `N` | New event |
| `T` | Go to today |
| `Left/Right` | Navigate dates |

## Navigation

| Shortcut | Action |
|----------|--------|
| `G then T` | Go to Today |
| `G then F` | Go to Focus |
| `G then P` | Go to Planner |
| `G then Q` | Go to Quests |
| `G then S` | Go to Settings |

## Inbox Mode

Access inbox from Command Palette:
1. Open Command Palette
2. Type note text
3. Press Enter to save to Infobase

## Tips

1. **Learn gradually** - Start with Command Palette
2. **Use often** - Muscle memory develops quickly
3. **Check tooltips** - Hover for shortcut hints
4. **Customize** - Settings for DAW-specific shortcuts
