# Today Dashboard

The Today page is your daily command center.

## Overview

Today shows:
- Personalized greeting
- Quick actions for common tasks
- Daily plan (if generated)
- Current streaks and stats

## Quick Actions

Organized into categories:

### Get Started
- **Focus** - Start a focus session
- **Planner** - View and manage calendar
- **Quests** - Track daily/weekly challenges
- **Exercise** - Log workouts

### Production
- **Shortcuts** - DAW keyboard shortcuts
- **Arrange** - Piano roll and drums
- **Reference** - Audio reference library
- **Templates** - Music production templates

### Learn & Grow
- **Learn** - Courses and lessons
- **Infobase** - Personal knowledge base
- **Goals** - Long-term objectives
- **Progress** - XP and skill stats

## Daily Plan

Generate a personalized daily plan:

1. Click "Plan My Day"
2. System creates a checklist:
   - 1 focus session
   - Active quests (up to 3)
   - Scheduled workout (if any)
   - Due review cards
   - Incomplete habits
3. Complete items from the plan
4. Track progress throughout the day

### Plan Items

Each item shows:
- Task type (icon)
- Title and description
- Action link
- Completion checkbox

## Navigation

From Today, access any feature:
- Click quick action cards
- Use sidebar navigation
- Open Command Palette (Cmd/Ctrl + K)
