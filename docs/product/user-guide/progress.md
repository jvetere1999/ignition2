# Progress & XP

Track your growth with the gamification system.

## Progress Dashboard

The Progress page shows:
- Current level and XP
- XP to next level
- Skill wheel visualization
- Coin balance
- Recent activities
- Streaks

## XP System

### Earning XP
Activities that earn XP:
| Activity | XP |
|----------|-----|
| Focus session | 25 |
| Workout complete | 50 |
| Lesson complete | 30 |
| Review session | 15 |
| Habit complete | 10 |
| Quest complete | Varies |
| Goal milestone | 40 |
| Planner task | 15 |

### Leveling Up

Level progression formula:
- Level 1: 0-99 XP
- Level 2: 100-299 XP (need 100 more)
- Level 3: 300-599 XP (need 200 more)
- Level 4: 600-999 XP (need 300 more)
- And so on...

Each level requires progressively more XP.

## Skill Wheel

Inspired by Persona 5, your XP distributes across five skills:

### Knowledge
- Focus sessions
- Learning activities
- Review sessions

### Guts
- Workouts
- Challenging tasks
- Facing fears

### Proficiency
- Production work
- Habits
- Planner tasks

### Kindness
- Community activities
- Helping others

### Charm
- Social tasks
- Presentations

The skill wheel visualizes your balance:
- Spokes represent each skill
- Outer ring = higher level
- Hover for details

## Coins

### Earning Coins
Coins are earned with XP:
- Ratio varies by activity
- Quests have fixed coin rewards

### Spending Coins
Use coins in the Market for rewards.

## Streaks

Track consistency:
- **Focus** - Days with focus sessions
- **Workout** - Days with logged workouts
- **Daily Activity** - Days with any activity

### Streak Protection
Purchase Streak Shields from Market:
- Protects one missed day
- Consumed when streak would break

## Recent Activities

Feed shows:
- Last 10 activities
- XP/coins earned
- Timestamps
