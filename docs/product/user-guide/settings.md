# Settings

Customize your Passion OS experience.

## Theme

Choose your visual theme:
- **Dark** - Dark background, easy on eyes
- **Light** - Bright and clean
- **System** - Matches your OS preference

## Account

Manage your account:
- View connected email
- Sign out
- Request account deletion

## Preferences

### DAW Preferences
Set your primary DAW for:
- Default shortcuts view
- Personalized tips

### Notification Preferences
Control notifications for:
- Focus reminders
- Break reminders
- Quest due alerts
- Workout reminders
- Review queue alerts

## Data

### Export Data
Download your data:
- Goals and progress
- Habits and logs
- Infobase entries
- Workout history

### Clear Local Data
Reset browser storage:
- Clears cached data
- Does not affect D1 database
- Use for troubleshooting

## Admin Console

For admin users only (visible if admin):
- User management
- Quest management
- Skill configuration
- System stats
