# Planner

A full-featured calendar for managing your schedule.

## Calendar Views

Switch between views:
- **Month** - Overview of the entire month
- **Week** - 7-day detailed view
- **Day** - Single day with hourly breakdown

## Creating Events

### Quick Add
Double-click any time slot to create an event.

### Full Form
1. Click "+ New Event"
2. Fill in details:
   - **Title** - Event name
   - **Type** - Meeting, Appointment, Workout, Other
   - **Date & Time** - Start and end
   - **Description** - Optional notes
   - **Recurrence** - Optional repeat pattern
3. Click "Save"

## Event Types

Events are color-coded:
- **Meeting** - Blue
- **Appointment** - Green
- **Workout** - Orange
- **Other** - Gray

## Recurring Events

Set events to repeat:
- **Daily** - Every day
- **Mon/Wed/Fri** - Three times per week
- **Tue/Thu/Sat** - Three times per week
- **Weekly** - Same day each week
- **Every 2 weeks** - Bi-weekly

## Managing Events

### Edit
Click an event to open the edit form.

### Delete
Click the delete button in the event details.

### Drag & Drop
Drag events to reschedule (in week/day view).

## Completing Tasks

When you complete a planner task:
- Click the complete button
- Earn XP and coins
- Task marked as done

Rewards:
- **15 XP** per completed task
- **5 coins** per completed task

## Integration

### From Exercise
Schedule workouts from the Exercise tab.

### From Programs
Training programs auto-populate the calendar.

### Notifications
Get reminders before scheduled events.

## Tips

1. **Block time** - Schedule focus sessions
2. **Include breaks** - Don't over-schedule
3. **Review weekly** - Plan ahead on Sundays
4. **Color code** - Use event types for visual clarity
