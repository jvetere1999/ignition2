# Production Tools

Tools for music producers and creators.

## Shortcuts Hub

Browse DAW keyboard shortcuts.

### Supported DAWs
- Ableton Live
- Logic Pro
- FL Studio
- Pro Tools
- Cubase
- Reaper
- Bitwig Studio
- Studio One
- Reason

### Features
- Category-based organization
- Search and filter
- Mac/PC keyboard toggle
- Visual keyboard display
- Mark shortcuts as learned

### Using Shortcuts
1. Go to **Shortcuts** (Hub)
2. Select your DAW
3. Browse by category or search
4. Toggle Mac/Windows as needed
5. Click shortcuts to see key display

## Arrange View

Interactive music workspace.

### Piano Roll
- Full octave range (C-1 to C6)
- Click to place notes
- Drag for sustain/length
- Velocity control

### Drum Machine
- Multiple drum sounds
- Grid-based sequencing
- Time-locked to grid
- Per-step velocity

### Controls
- **Play/Stop** - Playback controls
- **BPM** - Tempo adjustment
- **Grid** - 1/4, 1/8, 1/16, 1/32 notes
- **Bars** - 4 to 16 bar length

## Templates

Music production starting points.

### Categories
- **Drums** - Rhythm patterns
- **Chords** - Harmonic progressions
- **Melody** - Melodic ideas

### Using Templates
1. Browse template library
2. Preview patterns
3. Copy to your DAW

## Reference Tracks

Audio reference library for A/B comparison.

### Adding Tracks
1. Go to **Reference**
2. Click "Add Folder" or "Add Track"
3. Select audio files from your computer

### Analysis
Automatic analysis includes:
- **BPM** detection
- **Key** detection
- **Waveform** visualization

### Markers
Set marker points for:
- Intro/Outro
- Drops/Builds
- Sections to reference

### Player
- Full playback controls
- Seek with waveform
- Loop sections
- Volume control

Note: Reference tracks are stored locally (not uploaded to cloud).

## Infobase

Personal knowledge base for production notes.

### Creating Entries
1. Click "Add Entry"
2. Write your note
3. Choose category
4. Add tags
5. Save

### Categories
- Mixing
- Sound Design
- Music Theory
- Workflow
- Tips
- Resources

### Searching
- Full-text search
- Filter by category
- Filter by tags
