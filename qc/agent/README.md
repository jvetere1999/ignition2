# QC Agent Prompts

Test prompts for ChatGPT Atlas to execute human-like UI testing.

## Usage

1. Open ChatGPT with Atlas/browser capability
2. Provide the target URL (e.g., `https://ignition.ecent.online`)
3. Copy/paste the prompt from the relevant test file
4. Atlas will execute the test and report results

## Test Categories

| File | Purpose |
|------|---------|
| `01-auth-flow.md` | OAuth login/logout flows |
| `02-onboarding.md` | New user onboarding experience |
| `03-dashboard.md` | Main dashboard functionality |
| `04-navigation.md` | Navigation and routing |
| `05-responsive.md` | Mobile/tablet responsiveness |
| `06-accessibility.md` | A11y compliance checks |
| `07-error-states.md` | Error handling and edge cases |
| `08-performance.md` | Load times and perceived performance |
