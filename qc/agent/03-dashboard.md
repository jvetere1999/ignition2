# Dashboard Tests

## Test 1: Dashboard Load & Layout

```
Log into [URL] and navigate to the main dashboard.

1. Note how long the dashboard takes to load
2. Describe the main layout (header, sidebar, main content, etc.)
3. Identify the key sections/widgets visible
4. Check if there's a loading state or skeleton UI during load

Report:
- Approximate load time (fast <1s, medium 1-3s, slow >3s)
- Describe the layout structure
- What are the main UI components visible?
- Is there a loading indicator?
```
